## 一、核心理念升级

### 1.1 定位升级

```yaml
原定位:
  "AI产品开发平台，AI团队帮你开发产品"

升级定位:
  "AI产品工厂，帮你从想法到赚钱"

核心转变:
  从: AI高管"讨论"和"建议"
  到: AI高管在虚拟环境中"真正执行工作"，并持续帮你赚钱

类比:
  原来: 像请了一群顾问，给你出主意
  现在: 像拥有一个完整的数字团队，帮你干活+帮你运营+帮你赚钱
```

### 1.2 核心价值链


## 三、[新增] 多格式需求输入

### 3.1 功能概述

```yaml
问题:
  - 用户不会写需求文档
  - 用户有各种现成资料但不知道怎么用
  - 纯文字描述容易遗漏和误解

解决方案:
  支持用户上传任意格式的资料，AI自动理解并提取需求

支持格式:
  - 文档: PDF, Word, Markdown, TXT
  - 表格: Excel, CSV
  - 图片: PNG, JPG (截图、草图、设计稿)
  - 链接: 网页URL, Figma链接
```

### 3.2 用户界面

```
│  │                                                            │  │
│  │  我想做一个在线课程平台...                                │  │
│  │                                                            │  │
│  └───────────────────────────────────────────────────────────┘  │

│  📎 上传参考资料 (可选)                                         │
│                                                                  │

│  │                                                          │    │
│  │     拖拽文件到这里，或点击上传                          │    │
│  │                                                          │    │
│  │     支持: PDF, Word, Excel, 图片, 草图, 链接...        │    │
```

### 3.3 各格式处理示例

**示例1: PDF需求文档**
```yaml
输入: PRD.pdf (20页)
处理: Mistral OCR提取 → AI识别结构 → 提取功能列表
输出: "从PRD中提取到23项功能需求 (P0: 8项, P1: 10项, P2: 5项)"
```

**示例2: 竞品截图**
```yaml
输入: competitor.png
处理: Claude Vision分析 → 识别UI组件 → 识别设计风格
输出: "识别到电商首页，2列商品网格，简约白色系风格"
```

**示例3: 手绘草图**
```yaml
输入: sketch.jpg (手机拍的草图)
处理: 图像增强 → 识别手绘元素 → 理解页面结构
输出: "从草图中理解到3个页面：登录页、首页、个人中心"
```

**示例4: Excel功能清单**
```yaml
输入: features.xlsx
处理: 解析表格 → 识别列含义 → 提取功能和优先级
输出: "从Excel中识别到18个功能，已按优先级分类"
```

**优先级: P1 | 预计开发: 1周**

## 四、[新增] AI开发沙盒

### 4.1 功能概述

```yaml
功能描述:
  为每个项目创建独立的云端开发环境，AI高管在其中真正执行开发工作

核心组件:
  1. 独立代码仓库 (Git)
  2. 独立数据库实例 (MongoDB/PostgreSQL)
  3. 独立运行时环境 (Node.js/Python等)
  4. 实时预览地址 (自动生成的临时域名)
  5. 在线IDE界面 (VS Code Web)

用户可见内容:
  - 实时代码编辑过程
  - 终端命令执行日志
  - 测试运行结果
  - 预览页面实时刷新
```

### 4.2 沙盒生命周期

```yaml
创建: 用户支付后自动创建
运行: 开发期间持续运行
暂停: 用户暂停项目时 (释放CPU，保留存储)
保留: 项目完成后保留7天供用户检查
销毁: 7天后自动销毁（用户可下载全部内容）
```

**优先级: P1 | 预计开发: 3周**
