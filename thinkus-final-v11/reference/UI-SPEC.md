## 1. 设计原则

### 1.1 核心原则

```yaml
简洁优先:
  - 减少认知负担
  - 一屏一个主要任务
  - 避免过多选项

透明可信:
  - AI过程可见
  - 进度实时更新
  - 费用清晰透明

引导式体验:
  - 清晰的下一步
  - 适时的帮助提示
  - 防错设计

情感化设计:
  - 适当的动画反馈
  - 成就感的营造
  - 友好的错误提示
```

```yaml
主按钮:
  背景: primary (#6366F1)
  文字: 白色
  圆角: 8px
  内边距: 12px 24px
  悬浮: primary-dark
  禁用: gray-400

次按钮:
  背景: 透明
  边框: 1px solid gray-300
  文字: gray-700
  悬浮: gray-50

幽灵按钮:
  背景: 透明
  文字: primary
  悬浮: primary-light/10

尺寸:
  sm: 32px高
  md: 40px高 (默认)
  lg: 48px高
```

```yaml
默认:
  边框: 1px solid gray-300

  边框: 1px solid error
  背景: error/5

成功状态:
  边框: 1px solid success
```

## 4. 动画规格

### 4.1 页面转场

```yaml
进入:
  opacity: 0 → 1
  transform: translateY(20px) → translateY(0)
  duration: 300ms
  easing: ease-out

退出:
  opacity: 1 → 0
  duration: 200ms
```

### 4.2 元素动画

```yaml
按钮悬浮:
  transform: scale(1.02)
  duration: 150ms

卡片悬浮:
  transform: translateY(-4px)
  box-shadow: lg
  duration: 200ms

加载指示:
  旋转动画
  duration: 1000ms
  infinite
```

### 4.3 特殊动画

```yaml
完成庆祝:
  - 彩带从顶部飘落
  - 标题放大弹入
  - 成就徽章依次出现

进度更新:
  - 进度条平滑填充
  - 数字递增动画

专家发言:
  - 头像呼吸效果
  - 消息渐入
```

## 5. 响应式设计

### 5.1 断点

```yaml
sm: 640px   # 小屏手机
md: 768px   # 大屏手机/小平板
lg: 1024px  # 平板/小笔记本
xl: 1280px  # 桌面
2xl: 1536px # 大屏桌面
```

### 5.2 布局适配

```yaml
首页:
  桌面: 多列布局
  平板: 两列布局
  手机: 单列布局

创建页:
  桌面: 左右分栏 (6:4)
  平板: 上下布局
  手机: 纯对话，功能面板改为底部抽屉

仪表盘:
  桌面: 侧边栏固定
  平板: 侧边栏可收起
  手机: 底部导航栏
```

## 6. 无障碍设计

```yaml
对比度:
  - 文字对比度 >= 4.5:1
  - 大文字对比度 >= 3:1

键盘导航:
  - 所有交互元素可通过Tab访问
  - 焦点状态清晰可见
  - 快捷键支持

屏幕阅读器:
  - 所有图片有alt文本
  - 表单有label关联
  - 动态内容有aria-live

动画:
  - 尊重prefers-reduced-motion
  - 提供关闭动画选项
```
